This plugin calling Hubspot chat through a custom button. You can add chat opening to any HTML tag.

== Description ==

= Features of Insert Headers and Footers =

* Automatic opening of the chat window when a user button or link is clicked
* Works through CSS class or ID
* You can add a function to a DOM element
* You can specify multiple classes or ID

== Installation ==

1. Install Chat opener for Hubspot by uploading the `chat-opener` directory to the `/wp-content/plugins/` directory. (See instructions on <a href="http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/" rel="friend">how to install a WordPress plugin</a>.)
2. Activate Chat opener for Hubspot the `Plugins` menu in WordPress.
3. Going to the `Settings > Chat opener` menu.
4. Fill and save plugin settings
